向量叉乘

左手定则:
vec3 front, right;
vec3 up = cross(right, front);
向量right从左手手心穿过, 除拇指外的四根手指指向front方向,这时大拇指指向的就是向量up的方向

code

camera

包含的向量元素:
位置pos,方向front,右right, 俯仰角(Pitch)、偏航角(Yaw),移动速度(MovementSpeed), 
        灵敏度(MouseSensitivity)

实现的动作:
    向前移动(forward): 沿front方向移动
    向后移动(backward): 沿front反方向移动
    向右移动(right): 沿向量right移动
    向左移动(left): 沿向right反向移动
    沿x轴转动:pitch
    沿y轴转动:yaw
